# Indel Distance

Calculates the insertion/deletion (Indel) distance between two strings.

## Usage

``` r
indel_distance(s1, s2)
```

## Arguments

- s1:

  The first string.

- s2:

  The second string.

## Value

A numeric value representing the Indel distance.

## Examples

``` r
indel_distance("kitten", "sitting")
#> [1] 5
```
