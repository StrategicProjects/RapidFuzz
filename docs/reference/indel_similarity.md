# Indel Similarity

Calculates the insertion/deletion (Indel) similarity between two
strings.

## Usage

``` r
indel_similarity(s1, s2)
```

## Arguments

- s1:

  The first string.

- s2:

  The second string.

## Value

A numeric value representing the Indel similarity.

## Examples

``` r
indel_similarity("kitten", "sitting")
#> [1] 8
```
