# Apply Edit Operations to String

Applies edit operations to transform a string.

## Usage

``` r
editops_apply_str(editops, s1, s2)
```

## Arguments

- editops:

  A data frame of edit operations (type, src_pos, dest_pos).

- s1:

  The source string.

- s2:

  The target string.

## Value

The transformed string.
